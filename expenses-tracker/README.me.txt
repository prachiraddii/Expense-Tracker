# Expense Tracker Project

## Project Overview
This is a simple Expense Tracker application built using Python. It allows users to add, view, and manage their daily expenses in an organized manner. The project demonstrates basic CRUD (Create, Read, Update, Delete) operations and file handling in Python.

## Features
- Add new expenses with details like date, category, and amount.
- View all recorded expenses in a structured format.
- Edit existing expenses.
- Delete expenses if needed.
- Save and retrieve expenses from a local file for persistence.

## Technologies Used
- Python 3.x
- CSV or text file for data storage
- Standard Python libraries (os, datetime, etc.)
